# Majuyayamandiri React + Firebase Hosting

## 📦 Setup
1️⃣ Install dependencies:
```
npm install
```

2️⃣ Build project:
```
npm run build
```

3️⃣ Deploy to Firebase:
```
firebase deploy
```

Fitur tambahan:
- Routing dengan React Router
- Styling dengan CSS
- Galeri foto dan video
- Firebase Realtime Database untuk backend sederhana
